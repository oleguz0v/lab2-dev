Программа на Go, которая подключается к базе данных и выполняет несколько простых операций над ней.

Поднять бд и выполнить программу на Go: 
```
docker-compose up -d
```
Подключииться к базе данных: 
```
docker-compose exec db mysql -u root -p recordings
```
Система запросит пароль, который можно взять из файла db-password.

После запуска, вы окажетесь в окружении SQL, где можно делать запросы к базе
данных. Например: SELECT * FROM album
